# CalcPad
My custom calculator macropad, will be used to calculate.


BOM

- 16x Blank DSA Keycaps
- 4x M3x16 Bolt
- 4x M3 Heatset
- 1x XIAO RP2040
- 16x Cherry MX Switches
- 1x 3D printed case (top and bottom)
- 1x 0.91 inch OLED display
- 16x Through-hole 1N4148 Diodes
